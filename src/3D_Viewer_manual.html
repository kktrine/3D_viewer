1. Introduction

This is the implementation of the education project of School21 - 3D Viewer.

The program provide the ability to:

  1. Load a wireframe model from an obj file (vertices and surfaces list support only).
  2. Translate the model by a specified distance in relation to the X, Y, Z axes.
  3. Rotate the model by a specified angle relative to its X, Y, Z axes.
  4. Scale the model by a specified value.
  5. Customizing the type of projection (parallel and central).
  6. Setting the color, type and width of facet, color, display method and size of vertex.
  7. Background color selection.
  8. Saving captured (rendered) images as bmp and jpeg files.

2. Installation

The installation is done with the console command ’make install’. You must be in the ’src’ directory of this project for the command to work. The build will be placed in the ’build’ directory created in ’src’.

The uninstalling is done with the console command ’make uninstall’.

3. Visualization of models

The program allows you to upload 3d models in obj format. Visualization of only vertices and faces is supported. To load a model click ’open’ button and select the obj format file.

After loading the model you can specify the distance to move the model (relative to the x, y, z axes), the angle to rotate the model (relative to the x, y, z axes), or the value to scale the model.

It is possible to rotate the model with mouse.

4. Settings

After loading the model you can change:

  1. Type of projection (parallel and central).
  2. Color (black, yellow, purple), type (solid, dashed) and width (from 1 to 10) of facet
  3. Color (black, yellow, purple), display method (none, circle, square) and size (from 1 to 10) of vertex.
  4. Background color (black, white, green).

To apply the settings click the ’apply settings’ button.

To change the settings to default click the ’default settings’ button and apply the settings.

The settings selected before closing the program will be saved and loaded the next time you start.

5. Creating images

To save the image select the format (bmp or jpeg) and click ’save image’. The program will prompt you to enter a name for the image and choose extension.
